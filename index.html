Q. What does this activity show?

This activity shows that images are made of tiny dots called pixels. When an image is zoomed, pixels become visible. Letters and pictures are formed by arranging pixels.

Page 57

Observation:

The default foreground color is Black and the default background color is White.

a. Foreground color box changes the drawing color.
b. Background color box changes the background or fill color.

Page 58

Q. Edit menu commands available only in GIMP:

  • Fill with FG Color
  • Fill with BG Color
  • Fill with Pattern
  • Stroke Selection
  • Stroke Path

Reason:

These options are available only in GIMP because it is an image editing software, while Writer and Impress are used for text and presentations.
